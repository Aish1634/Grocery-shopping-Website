# Grocery Shopping Web Application

## Description
A full-stack grocery shopping web application that allows users to browse products, view categories, and add items to a cart. The application is built using Node.js and Express for the backend, MongoDB for data storage, and HTML, CSS, and JavaScript for the frontend.

## Features
- Display grocery products with categories
- Add items to cart
- View cart items on a separate page
- Responsive user interface
- Backend API for product and cart management

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express  
- Database: MongoDB
